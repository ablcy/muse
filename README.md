# Muse - 灵感记录便签

Muse 是一个极简的灵感记录便签应用，帮助你在灵感闪现时快速捕捉。

## 功能

- 快速记录灵感（支持 Ctrl+Enter 快捷提交）
- 本地存储，数据完全保存在浏览器中
- 搜索便签内容
- 按时间筛选（全部 / 今日 / 本周）
- 一键复制便签内容
- 响应式设计，适配桌面和移动端

## 技术栈

纯前端实现：HTML + CSS + JavaScript，无需任何依赖。

## 使用方式

直接在浏览器中打开 `index.html` 即可使用。

## 项目结构

```
muse/
├── index.html    # 主页面
├── style.css     # 样式文件
└── script.js     # 交互逻辑
```
